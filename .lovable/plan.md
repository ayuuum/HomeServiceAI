
# ランディングページ B2B向け改修計画

## 現状の問題

現在のLPは「ハウスクリーニングを受けたい一般消費者（B2C）」向けの内容になっています：
- エアコン、キッチン、浴室のクリーニングサービス紹介
- 「今すぐ予約する」CTA
- 一般消費者向けの訴求

## あるべき姿

「ハウクリPro」という予約管理SaaSを導入したいハウスクリーニング事業者（B2B）向けのLPに変更します。

## 変更内容

### 変更ファイル: `src/pages/LandingPage.tsx`

#### ヒーローセクション
**変更前：**
- キャッチコピー：「プロの技術で、暮らしをもっと快適に」
- CTA：「今すぐ予約する」→ `/booking/default`

**変更後：**
- キャッチコピー：「ハウスクリーニング業務を、もっとスマートに」
- サブコピー：「予約受付・顧客管理・スケジュール調整をワンストップで」
- CTA：「お問い合わせ」→ お問い合わせセクションへスクロール

#### サービス紹介 → 機能紹介セクション
**変更前：** エアコン、キッチン、浴室クリーニングの紹介

**変更後：** システム機能の紹介
| 機能 | アイコン | 説明 |
|------|---------|------|
| オンライン予約受付 | Calendar | お客様が24時間いつでもWeb予約。電話対応の手間を削減 |
| 顧客管理（CRM） | Users | 顧客情報・予約履歴を一元管理。リピート促進に活用 |
| スタッフ管理 | UserCheck | スタッフのスケジュールをガントチャートで可視化 |
| 売上レポート | BarChart3 | 売上・予約数を自動集計。経営判断をサポート |

#### 選ばれる理由 → 導入メリットセクション
**変更前：** プロの技術、安心価格、簡単予約

**変更後：**
| メリット | 説明 |
|---------|------|
| 予約業務を効率化 | 電話・メール対応を削減。予約確認・リマインドも自動化 |
| 24時間自動受付 | 営業時間外も予約を逃さない。機会損失を防止 |
| 顧客満足度向上 | スムーズな予約体験で、リピート率アップに貢献 |

#### CTAセクション
**変更前：** 「今すぐ予約しませんか？」→「予約ページへ」

**変更後：** 
- 見出し：「導入のご相談はこちら」
- 説明：「お気軽にお問い合わせください。デモのご案内も可能です。」
- CTA：「お問い合わせ」→ メール（`mailto:contact@haukuripro.com`）またはLINE連携

## アイコンの変更

**削除するアイコン：**
- Wind, ChefHat, Bath（クリーニングサービス用）

**追加するアイコン：**
- Calendar（予約管理）
- Users（顧客管理）
- UserCheck（スタッフ管理）
- BarChart3（レポート）
- Mail（お問い合わせ）

## 技術的詳細

### コンポーネント構成
- `FeatureCard`コンポーネントを再利用（機能紹介用）
- `ServiceCard`コンポーネントは削除（サービス画像が不要になるため）
- 新しい`FeatureCard`はアイコン＋タイトル＋説明のシンプルな構成

### CTAの実装
- ページ内スクロール（`#contact`アンカー）
- メールリンク（`mailto:contact@haukuripro.com`）
- 将来的にお問い合わせフォームを追加する場合は別ページ化も可能

### デザイン方針
- 既存のブランドカラー（ダークネイビー）を維持
- モバイルファースト対応
- クリーン＆信頼感のあるB2B SaaSらしいデザイン
